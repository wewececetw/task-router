翻譯 i18n 字串檔。

步驟：
1. 用 Read 讀取來源語言的 i18n 檔案（如 `en.json`）
2. 用 Bash 執行 `./scripts/call-omlx.sh` 翻譯：

```bash
./scripts/call-omlx.sh "將以下 i18n JSON 翻譯成 {目標語言}。
規則：
- 保持 key 不變，只翻譯 value
- 保留 {{variable}} 和 {0} 等佔位符
- 保持 JSON 格式
- 用詞自然口語化

{i18n JSON 內容}" --max-tokens 4096
```

3. 從輸出中提取翻譯後的 JSON，用 Write 寫入對應的語言檔案
4. 如果 helper 回傳 FALLBACK，你自己翻譯

$ARGUMENTS
